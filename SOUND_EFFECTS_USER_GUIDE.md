# 🔊 효과음 시스템 사용자 가이드

## 🎵 효과음이란?

한국ESG학회 웹사이트는 이제 **인터랙티브한 사운드 효과**를 제공합니다!

---

## ✨ 두 가지 효과음

### 1. 📍 호버 효과음 (스륵)
- **언제**: 메뉴에 마우스를 올렸을 때
- **소리**: 부드러운 "스륵" 소리
- **목적**: 메뉴 탐색 시 미묘한 피드백

### 2. 🖱️ 클릭 효과음 (딸깍)
- **언제**: 버튼이나 링크를 클릭했을 때
- **소리**: 명확한 "딸깍" 소리
- **목적**: 클릭 동작 확인 피드백

---

## 🎛️ 효과음 켜고 끄기

### 토글 버튼 위치
- 화면 **우측 하단**에 **보라색 원형 버튼** 🟣
- 스크롤 상단 버튼 바로 위

### 상태 확인
- 🔊 **소리 아이콘**: 효과음 켜짐
- 🔇 **음소거 아이콘**: 효과음 꺼짐
- 마우스를 올리면 **"효과음 켜짐/꺼짐"** 툴팁 표시

### 작동 방법
1. 보라색 버튼 클릭
2. 아이콘이 즉시 변경됨
3. 설정이 자동으로 저장됨
4. 페이지를 새로고침해도 설정 유지

---

## 💡 팁

### 효과음이 들리지 않나요?
- ✅ 브라우저 음소거 해제 확인
- ✅ 시스템 볼륨 확인
- ✅ 우측 하단 버튼이 🔊 상태인지 확인
- ✅ 모던 브라우저 사용 (Chrome, Firefox, Safari, Edge)

### 효과음이 너무 시끄러운가요?
- 🔇 우측 하단 버튼으로 끄기
- 또는 시스템 볼륨 조절

### 설정이 저장되나요?
- ✅ 네! LocalStorage에 자동 저장됩니다
- ✅ 브라우저를 닫았다가 다시 열어도 유지됩니다
- ✅ 같은 브라우저 내에서 모든 페이지에 적용됩니다

---

## 🎯 효과음 적용 범위

효과음이 작동하는 요소들:

### 호버 효과음 (스륵)
- 상단 메뉴 링크
- 드롭다운 메뉴 아이템

### 클릭 효과음 (딸깍)
- 모든 버튼
- 모든 메뉴 링크
- 카드 링크
- 스크롤 상단 버튼
- 슬라이더 버튼
- 검색 버튼
- 로그인/회원가입 버튼
- 기타 모든 클릭 가능한 요소

---

## 🔧 개발자 도구

F12 개발자 콘솔에서 사용 가능:

```javascript
// 효과음 토글
toggleSoundEffects()

// 효과음 상태 확인 (true/false)
getSoundEffectsStatus()

// 전역 soundEffects 객체 접근
window.soundEffects.playHover()  // 호버 소리 재생
window.soundEffects.playClick()  // 클릭 소리 재생
```

## 🌟 왜 효과음을 추가했나요?

### 사용자 경험 향상
- 🎯 **피드백**: 클릭이 제대로 되었는지 즉시 확인
- 🎨 **몰입감**: 더 생동감 있는 웹 경험
- ♿ **접근성**: 시각 외 청각 피드백 제공
- 🎪 **재미**: 인터랙티브하고 현대적인 느낌

### 웹 표준 준수
- ✅ Web Audio API 사용
- ✅ Progressive Enhancement
- ✅ 성능 최적화
- ✅ 사용자 제어 가능
