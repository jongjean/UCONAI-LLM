## 🎯 배포 방법 (3단계)

### 1단계: Publish 탭 접속
1. 화면 상단의 **"Publish"** 탭 클릭
2. 프로젝트 배포 화면으로 이동

### 2단계: 프로젝트 배포
1. **"Publish Project"** 또는 **"Deploy"** 버튼 클릭
2. 자동 빌드 및 배포 프로세스 시작
3. 완료까지 약 2-3분 소요

### 3단계: URL 확인 및 테스트
1. 배포 완료 후 제공되는 **공개 URL 복사**
2. 새 탭에서 URL 접속하여 정상 작동 확인
3. 시연에 사용할 URL 저장

**예상 URL 형식:**
```
https://[project-id].web.app
또는
https://[custom-domain].vercel.app
```

## 🔧 문제 해결 가이드

### 문제 1: 배포가 실패함
**원인:** 파일 크기 초과, 문법 오류 등  
**해결:**
1. 콘솔 오류 메시지 확인
2. HTML/CSS/JS 문법 오류 수정
3. 이미지 파일 크기 최적화
4. 재배포 시도

### 문제 2: CSS가 적용되지 않음
**원인:** 경로 오류, 캐시 문제  
**해결:**
1. `Ctrl + Shift + R` (강력 새로고침)
2. CSS 파일 경로 확인 (`css/style.css`)
3. 브라우저 캐시 완전 삭제
4. 시크릿 모드에서 테스트

### 문제 3: JavaScript가 작동하지 않음
**원인:** 스크립트 로딩 오류, 문법 오류  
**해결:**
1. `F12` → Console 탭에서 오류 확인
2. JavaScript 파일 경로 확인
3. 외부 CDN 로딩 확인 (Font Awesome 등)

### 문제 4: 결제 시스템이 작동하지 않음
**원인:** LocalStorage 문제, 스크립트 오류  
**해결:**
1. `F12` → Console → `localStorage.clear()` 실행
2. 새로고침 후 재시도
3. 다른 브라우저에서 테스트

### 문제 5: 이미지가 표시되지 않음
**원인:** 이미지 경로 오류, 파일 누락  
**해결:**
1. 이미지 경로 확인 (`images/logo.png`)
2. 파일이 실제로 배포되었는지 확인
3. 이미지 URL 절대 경로로 변경 고려

시연 전 성능을 확인하고 싶다면:
