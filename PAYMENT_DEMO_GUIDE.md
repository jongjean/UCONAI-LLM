# 💳 카드결제 데모 사용 가이드

## 🎯 빠른 시작

### 1. 페이지 접속
```
pages/mypage/payment.html
```

### 2. 테스트 카드 정보 입력

#### 카드번호
```
1234 5678 9012 3456
```

#### 유효기간
```
MM: 12
YY: 25
```

#### CVC
```
123
```

#### 카드 소유자 이름
```
홍길동
```

#### 카드사
```
신한카드 (또는 원하는 카드사 선택)
```

---

## ✨ 주요 기능

### ✅ 데모 모드 동작
1. 모든 카드 정보 입력
2. "결제 진행에 동의합니다" 체크
3. "결제하기" 버튼 클릭
4. 2초 로딩 애니메이션
5. 결제 완료 메시지 표시

### ✅ 실시간 카드 비주얼
- 카드번호 입력 → 카드에 실시간 표시
- 유효기간 입력 → 카드에 MM/YY 표시
- 이름 입력 → 대문자로 변환되어 표시
- 카드사 선택 → 카드 상단에 표시

### ✅ 자동 포커스
- 카드번호 4자리 입력 → 자동으로 다음 칸 이동
- 유효기간 월 입력 (2자리) → 자동으로 연도 칸 이동

---

## 🚨 실제 결제 주의사항

### ⚠️ 현재는 데모 모드입니다
- 실제 결제는 **백엔드 API 연동 후** 가능
- 입력한 카드 정보는 **어디에도 저장되지 않습니다**
- 결제 금액은 **청구되지 않습니다**

### 🔌 실제 결제를 위해 필요한 것
1. **백엔드 서버** (Node.js, Python, Java 등)
2. **PG사 계약** (토스페이먼츠, 아임포트 등)
3. **API 키** (PG사에서 발급)
4. **HTTPS 인증서** (SSL)

자세한 연동 방법은 `PAYMENT_IMPLEMENTATION_GUIDE.md` 참고

## 🐛 문제 해결

### Q: 결제 버튼이 비활성화되어 있어요
A: "결제 진행에 동의합니다" 체크박스를 체크하세요.

### Q: 카드번호 입력이 안 돼요
A: 숫자만 입력 가능합니다. (영문, 특수문자 X)

### Q: 유효기간 월에 13을 입력했는데 12로 바뀌어요
A: 정상 동작입니다. 월은 01~12만 유효합니다.

### Q: 실제로 결제가 되나요?
A: 아니요. 현재는 데모 모드입니다. 백엔드 API 연동 후 실제 결제 가능합니다.
