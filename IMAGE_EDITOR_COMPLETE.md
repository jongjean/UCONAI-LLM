# ✅ 포스팅툴 이미지 편집 기능 추가 완료!

## 🎨 새로운 기능

### 1. **이미지 편집기** 🖼️
- 확대/축소 (50% ~ 200%)
- 상하좌우 드래그 이동
- 버튼으로 정밀 이동
- 실시간 미리보기

### 2. **드래그 앤 드롭** 📎
- 이미지 파일을 드래그해서 업로드
- 포스팅툴 이미지 영역에 드롭

### 3. **복사 & 붙여넣기** 📋
- 이미지를 복사 (Ctrl+C)
- 포스팅툴에서 붙여넣기 (Ctrl+V)
- 즉시 업로드

### 4. **마우스 휠 줌** 🖱️
- 이미지 위에서 마우스 휠 → 확대/축소

### 5. **키보드 컨트롤** ⌨️
- 화살표 키 → 이미지 이동
- 정밀한 위치 조정

## 🎯 사용 방법

### **이미지 편집하기**

1. **포스팅툴 접속**
   - `pages/admin/posting-tool.html`

2. **슬라이드 카드에서 이미지 클릭**
   - 이미지 위의 **"편집"** 버튼 (자르기 아이콘) 클릭

3. **이미지 편집 모달 열림**
   - 왼쪽: 편집 캔버스 (800x450px)
   - 오른쪽: 컨트롤 패널

4. **이미지 조정**
   
   #### 방법 1: 드래그
   - 이미지를 마우스로 드래그해서 이동
   
   #### 방법 2: 줌 슬라이더
   - 슬라이더를 움직여서 확대/축소 (50% ~ 200%)
   
   #### 방법 3: 방향 버튼
   - 상하좌우 화살표 버튼으로 정밀 이동
   - 중앙 버튼으로 초기화
   
   #### 방법 4: 마우스 휠
   - 이미지 위에서 마우스 휠 → 확대/축소
   
   #### 방법 5: 키보드
   - 화살표 키로 이미지 이동

5. **편집 완료**
   - **"적용"** 버튼 클릭
   - localStorage에 자동 저장
   - 메인 페이지에 즉시 반영

---

### **드래그 앤 드롭으로 이미지 업로드**

1. **PC에서 이미지 파일 찾기**

2. **이미지를 포스팅툴로 드래그**
   - 슬라이드 카드의 이미지 영역으로 드래그

3. **드롭!**
   - 이미지가 즉시 업로드됨
   - localStorage에 Base64로 저장

---

### **복사 & 붙여넣기로 이미지 업로드**

1. **이미지 복사**
   - 웹 브라우저에서 이미지 우클릭 → "이미지 복사"
   - 또는 이미지 파일 탐색기에서 복사

2. **포스팅툴에서 붙여넣기**
   - `Ctrl + V` (Windows)
   - `Cmd + V` (Mac)

3. **즉시 업로드**
   - 이미지가 슬라이드에 적용됨

## 🎬 편집 예시

### 시나리오 1: 이미지 확대 & 중앙 맞추기

```
1. 포스팅툴 접속
2. 슬라이드 이미지에서 "편집" 버튼 클릭
3. 줌 슬라이더를 150%로 조정
4. 이미지를 드래그해서 중요한 부분을 중앙에 배치
5. "적용" 클릭
6. 메인 페이지 새로고침 → 확대된 이미지 표시!
```

### 시나리오 2: 드래그 앤 드롭 업로드

```
1. PC에서 이미지 파일 선택
2. 포스팅툴 슬라이드 카드로 드래그
3. 드롭!
4. 즉시 업로드 완료
5. 필요하면 "편집" 버튼으로 위치 조정
```

### 시나리오 3: 복사 & 붙여넣기

```
1. Google 이미지 검색에서 원하는 이미지 찾기
2. 이미지 우클릭 → "이미지 복사"
3. 포스팅툴로 이동
4. Ctrl + V
5. 즉시 업로드 완료!
```

## 🔧 기술 세부사항

### 이미지 변형 저장 방식

```javascript
// 포스팅툴에서 저장
slide.imageTransform = {
    zoom: 150,          // 150%
    positionX: -50,     // 좌우 이동
    positionY: 30       // 상하 이동
};
```

### 메인 페이지에서 적용

```javascript
// main.js에서 변형 적용
if (slide.imageTransform) {
    const transform = slide.imageTransform;
    const scale = transform.zoom / 100;
    slideElement.style.backgroundSize = `${scale * 100}%`;
    slideElement.style.backgroundPosition = 
        `${50 + (transform.positionX / 8)}% ${50 + (transform.positionY / 4.5)}%`;
}
```
