# ⚡ 빠른 시연 가이드 (5분 완성)

**배포 전 필수 체크!** ✅

---

## 🚀 30초 배포 방법

### 1단계: Publish 탭 클릭
- 화면 상단의 **"Publish"** 탭 클릭

### 2단계: 배포 버튼 클릭
- **"Publish Project"** 버튼 클릭

### 3단계: URL 복사
- 배포 완료 후 제공되는 **URL 복사**
- 이 URL로 시연 진행!

---

## 🎯 3분 시연 시나리오

### 1️⃣ 메인 페이지 (30초)
- 슬라이드 배너 자동 재생 확인
- "ESG는 지속가능한 미래를 만듭니다"
- 퀵 링크 카드 (학회 소개, 회원가입, 학술지)

### 2️⃣ 메뉴 네비게이션 (30초)
- 상단 메뉴 호버 → 드롭다운 표시
- "학회소개" → "인사말" 클릭
- 사운드 효과 (토글 버튼 우측 하단)

### 3️⃣ **결제 시스템** (2분) ⭐ 가장 중요!
1. "마이페이지" → "회비·납부 내역" 클릭
2. "신용카드 결제" 선택
3. "결제하기" 버튼 클릭
4. 모달에서 정보 입력:
   - 카드번호: `1234-5678-9012-3456`
   - 유효기간: `12/25`
   - CVC: `123`
   - 이름: `홍길동`
5. "결제 진행" 클릭
6. ✅ 결제 완료 페이지 표시!
7. "납부 내역 확인" 클릭
8. 📊 자동으로 통계 및 내역 표시

## 📱 예상 질문 & 간단 답변

**Q: 실제 결제가 되나요?**
> A: 시뮬레이션이며, PG사 API만 추가하면 실결제 가능합니다.

**Q: 언제 배포 가능한가요?**
> A: 지금 즉시 가능합니다. Publish 탭에서 원클릭 배포!

**Q: 추가 개발이 필요한가요?**
> A: 기본 구조는 완성. 실제 콘텐츠 업데이트와 백엔드 연동만 필요합니다.

## 🔥 긴급 상황 대응

**문제: 페이지 안 열림**
- 해결: `Ctrl + Shift + R` (캐시 클리어 새로고침)

**문제: 결제 안 됨**
- 해결: F12 콘솔 → `localStorage.clear()` 실행 후 새로고침
