## 🎨 디자인 하이라이트

### 사이드바 스타일
```
- 배경: 화이트 (#ffffff)
- 헤더: 그린-블루 그라데이션
- 보더: 부드러운 라운드 (8px)
- 그림자: 은은한 섀도우 (0 2px 8px rgba(0,0,0,0.08))
- 활성 메뉴: 좌측 녹색 보더 (4px) + 연녹색 배경
```

### 인터랙션 효과
```
- 호버: 왼쪽으로 5px 이동 + 배경색 변경
- 화살표: 호버/활성 시 '›' 기호 표시
- 모바일: 터치로 펼침/접힘 토글
```

### 브레드크럼
```
- 위치: 페이지 상단, 사이드바 위
- 아이콘: Font Awesome 화살표 사용
- 현재 페이지: 녹색 강조
```

## 📝 사용 방법

### 1. 기본 HTML 구조
```html
<!-- CSS 포함 -->
<link rel="stylesheet" href="../../css/sidebar.css">

<!-- 페이지 레이아웃 -->
<div class="page-layout">
    <!-- 사이드바 -->
    <aside class="sidebar">
        <div class="sidebar-title">메뉴 제목</div>
        <ul class="sidebar-menu">
            <li><a href="#" class="active">메뉴 1</a></li>
            <li><a href="#">메뉴 2</a></li>
        </ul>
    </aside>

    <!-- 메인 콘텐츠 -->
    <main class="main-content">
        <h2 class="content-title">페이지 제목</h2>
        <!-- 내용 -->
    </main>
</div>
```

### 2. 모바일 토글 스크립트
```javascript
document.addEventListener('DOMContentLoaded', function() {
    const sidebarTitle = document.querySelector('.sidebar-title');
    const sidebar = document.querySelector('.sidebar');
    
    if (window.innerWidth <= 768) {
        sidebarTitle.addEventListener('click', function() {
            sidebar.classList.toggle('collapsed');
        });
    }
});
```
