# 컨셉 컬러 적용 - 2025-12-27

## 🎯 작업 내용

ESG 테마에 맞춰 홈페이지 전체에 **컨셉 컬러 적용**

## ✅ 컬러 팔레트 추가

### 신규 컬러 변수
```css
:root {
    --bg-lighter: #ffffff;              /* 순백 */
    --bg-green-light: #e8f5e9;          /* 연한 초록 */
    --bg-blue-light: #e3f2fd;           /* 연한 파랑 */
    --bg-gradient: linear-gradient(135deg, #e8f5e9 0%, #e3f2fd 100%);
}
```

### 기존 컬러
```css
--primary-green: #1e7e34;              /* ESG 초록 */
--primary-blue: #2874a6;               /* ESG 파랑 */
--bg-light: #f8f9fa;                   /* 연한 회색 */
```

## 📐 디자인 강화

### 1. 섹션 타이틀 밑줄
```css
.section-title::after {
    content: '';
    width: 80px;
    height: 4px;
    background: linear-gradient(90deg, 초록, 파랑);
}
```

**효과**: 
```
    ESG 가치
    ────
    (그라데이션)
```

```css

```

## 🌈 컬러 컨셉

### ESG 연결
- **Environment (환경)**: 초록 계열 (#e8f5e9)
- **Social (사회)**: 파랑 계열 (#e3f2fd)
- **Governance (지배구조)**: 회색/흰색 계열

### 시각적 리듬
```
┌─────────────────────────────┐
│ Header (흰색)               │ ← 깔끔
├─────────────────────────────┤
│ Hero (그라데이션)           │ ← 역동적
├─────────────────────────────┤
│ ESG Values (연한 초록)      │ ← 환경
├─────────────────────────────┤
│ News (흰색)                 │ ← 휴식
├─────────────────────────────┤
│ Quick Links (연한 파랑)     │ ← 신뢰
├─────────────────────────────┤
│ Footer (어두운 청회색)       │ ← 안정
└─────────────────────────────┘
```

## 🎯 디자인 원칙

### 1. 가독성 우선
- 배경색: 연한 파스텔 톤
- 텍스트: 충분한 대비 유지
- 카드: 흰색 배경으로 내용 강조

### 2. ESG 정체성
- 초록: 환경 (Environment)
- 파랑: 사회적 책임 (Social)
- 회색: 투명성 (Governance)

### 3. 시각적 계층
- 헤더: 고정, 흰색 (최상위)
- 섹션: 교차 배경 (구분)
- 카드: 흰색 + 그림자 (강조)
- 푸터: 어두움 (마무리)

## 💡 개선 효과

- ✅ **구분감**: 섹션별로 명확한 시각적 구분
- ✅ **전문성**: 세련되고 현대적인 느낌
- ✅ **브랜드**: ESG 컨셉 색상 활용
- ✅ **가독성**: 흰색 카드로 내용 강조
- ✅ **역동성**: 단조롭지 않은 레이아웃
