## 🎨 스타일 변경 요약

### HTML (index.html)
```html
<!-- Before -->
<div class="top-bar">
    <div class="contact-info">
        <span>전화번호</span>
        <span>이메일</span>
    </div>
</div>
<a href="index.html" class="logo">
    <img src="images/logo-full.png" class="logo-full">
    <img src="images/logo-symbol.png" class="logo-symbol">
</a>

<!-- After -->
<a href="index.html" class="logo">
    <img src="images/logo-full.png" class="logo-full">
</a>
```

### CSS (css/style.css)
```css
/* 로고 크기 증가 */
.logo-full {
    height: 55px;  /* 기존: 45px */
}

/* 심볼 로고 숨김 */
.logo-symbol {
    display: none;
}

/* 네비게이션 패딩 증가 */
.navbar {
    padding: 15px 0;  /* 기존: 10px 0 */
}

/* 메뉴 텍스트 크기 증가 */
.nav-link {
    font-size: 0.9rem;     /* 기존: 0.8rem */
    padding: 10px 14px;    /* 기존: 8px 8px */
}

/* 모바일 전환 시점 변경 */
@media (max-width: 900px) {  /* 기존: 1100px */
    /* 모바일 메뉴 */
}
```
