# 메뉴 간격 최소화 - 2025-12-27

## 🎯 사용자 요청
메뉴 간격이 너무 넓음 → **최대한 다 붙여서 다닥다닥**

## ✅ 최종 수정

### 1. 메뉴 간격 최소화
- **gap**: 8px → **2px** (거의 붙음!)
- **padding**: 8px 14px → **8px 10px** (좌우 여백 축소)
- **font-size**: 0.8rem → **0.75rem** (더 작게)
- **border-radius**: 6px → **4px** (더 타이트)
- **화살표 간격**: 4px → **3px**
- **화살표 크기**: 10px → **9px**

### 2. 최종 CSS
```css
.nav-menu {
    gap: 2px;                      /* 거의 붙음! */
}

.nav-link {
    padding: 8px 10px;             /* 컴팩트 */
    font-size: 0.75rem;            /* 작은 글씨 */
    border-radius: 4px;            /* 샤프한 모서리 */
}

.has-dropdown .nav-link::after {
    margin-left: 3px;              /* 화살표 가까이 */
    font-size: 9px;                /* 작은 화살표 */
}
```

### 3. 결과
- ✅ 메뉴가 **다닥다닥** 붙어 있음
- ✅ 11개 메뉴가 한 줄에 모두 표시
- ✅ 공간 효율 극대화
- ✅ 여전히 버튼 스타일 유지
