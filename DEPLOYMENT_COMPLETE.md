## 🎯 시연 시나리오

### 1단계: 메인 페이지 (30초)
**URL:** `/index.html`
- 슬라이드 배너 확인
- 퀵 링크 클릭
- ESG 가치 소개

### 2단계: 메뉴 네비게이션 (30초)
- 상단 메뉴 호버
- 드롭다운 확인
- 하위 페이지 이동

### 3단계: 결제 시스템 ⭐ (2분)
**URL:** `/pages/mypage/payment.html`

**시연 플로우:**
1. 마이페이지 → 회비·납부 내역
2. 신용카드 결제 선택
3. 결제 정보 입력:
   - 카드번호: `1234-5678-9012-3456`
   - 유효기간: `12/25`
   - CVC: `123`
   - 이름: `홍길동`
4. 결제 진행 클릭
5. 결제 완료 페이지 확인
6. 납부 내역 확인

## 📞 예상 질문 & 답변

**Q1: 실제 결제가 가능한가요?**
> A: 현재는 시뮬레이션 모드이며, 포트원이나 토스페이먼츠 같은 PG사 API를 연동하면 즉시 실결제가 가능합니다. 구조는 이미 완성되어 있습니다.

**Q2: 회원 데이터는 어떻게 관리하나요?**
> A: 현재는 LocalStorage로 프론트엔드에 저장되며, 향후 백엔드 API와 데이터베이스(MySQL, PostgreSQL 등)로 전환 가능합니다.

**Q3: 추가 개발이 필요한가요?**
> A: 기본 구조는 100% 완성되었습니다. 실제 콘텐츠 입력, 백엔드 연동, 실결제 연동만 추가하면 됩니다.

**Q4: 모바일에서도 잘 작동하나요?**
> A: 네, 완전한 반응형 디자인으로 스마트폰, 태블릿, 데스크톱 모두 최적화되어 있습니다.

**Q5: 얼마나 빨리 실운영이 가능한가요?**
> A: 지금 즉시 가능합니다. 실제 콘텐츠만 입력하면 바로 운영할 수 있습니다.
