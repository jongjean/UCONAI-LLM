## 🎯 관리자 메뉴 접근

### **방법 1: 마이페이지를 통한 접근** (정식 경로)

**1단계**: 로그인 완료 후 우측 상단 "마이페이지" 클릭
```
https://www.genspark.ai/api/code_sandbox_light/preview/68d5a3b6-99a3-44d6-8a91-440bc5253b4c/pages/mypage/profile.html?canvas_history_id=bd8255b8-e628-4063-9269-b2a901665fe9
```

**2단계**: 좌측 사이드바에서 "관리자 메뉴" 섹션 찾기
- 📍 위치: "빠른 메뉴" 아래
- 🎨 아이콘: 👑 왕관
- 🎨 배경: 녹색

**3단계**: 원하는 도구 선택
- **📝 포스팅툴**: 히어로 슬라이더 편집
- **🕐 히스토리 관리**: 버전 관리

---

### **방법 2: 테스트 페이지 활용** (⚡ 가장 빠름)

**1단계**: 테스트 페이지 직접 접속
```
https://www.genspark.ai/api/code_sandbox_light/preview/68d5a3b6-99a3-44d6-8a91-440bc5253b4c/posting-tool-access.html?canvas_history_id=bd8255b8-e628-4063-9269-b2a901665fe9
```

**2단계**: "포스팅툴 열기" 또는 "히스토리 관리 열기" 버튼 클릭

**3단계**: 관리자 도구 사용 시작!

**장점**:
- ✅ 자동 로그인 처리
- ✅ 모든 관리자 링크 한 곳에 모음
- ✅ 클릭 한 번으로 접근
- ✅ 5초 만에 포스팅툴 도달

---

### **방법 3: 브라우저 콘솔에서 즉시 로그인** (개발자용)

**1단계**: F12 눌러서 콘솔 열기

**2단계**: 아래 코드 복사해서 붙여넣기
```javascript
// 관리자 계정으로 자동 로그인
localStorage.setItem('user', JSON.stringify({
    id: 'test@esg.or.kr',
    name: '관리자',
    role: 'admin',
    loginTime: new Date().toISOString()
}));

// 페이지 새로고침
location.reload();
```

**3단계**: Enter 키 누르기

**4단계**: 페이지가 새로고침되면서 자동 로그인 완료! ✅

## 🔧 문제 해결

### ❓ 로그인 버튼이 보이지 않을 때
**원인**: 이미 로그인된 상태
**해결**: 우측 상단에 사용자 이름 확인

### ❓ 로그인 후 관리자 메뉴가 보이지 않을 때
**원인**: 마이페이지에 가지 않음
**해결**: 우측 상단 "마이페이지" 클릭 → 좌측 사이드바 확인

### ❓ 비밀번호를 잊어버렸을 때
**해결**: 테스트 페이지 사용 또는 콘솔에서 즉시 로그인

### ❓ 로그인이 계속 실패할 때
**확인 사항**:
- ✅ 아이디: `test@esg.or.kr` (철자 확인)
- ✅ 비밀번호: `password123` (대소문자 구분)
- ✅ 임시 계정: 비밀번호 6자 이상
